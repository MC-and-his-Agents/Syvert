- 发布 `#445` Batch / Dataset Core Contract release slice，完成 batch request / target set、item-level outcome、partial success/failure、resume token、batch audit trace、dataset record / sink 最小模型、readback 与 audit replay。

## 明确不在范围

- 不交付 scheduler / scheduled execution。
